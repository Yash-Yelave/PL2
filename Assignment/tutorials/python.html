Python Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. It's widely used in web development, data science, artificial intelligence, and more.

Example:

# Simple Python program
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))  # Output: Hello, World!

Python Syntax

Python syntax is known for its readability and simplicity. It uses indentation to define code blocks instead of curly braces.

Example:

# Python uses indentation to define blocks
if 5 > 2:
    print("Five is greater than two!")
    print("This is part of the if block")

# This is outside the if block
print("This is always printed")

Python Variables

Variables in Python are created when you assign a value to them. Python is dynamically typed, so you don't need to declare the type of a variable.

Example:

# Variable assignment
x = 5
y = "Hello"
z = 3.14

# Multiple assignment
a, b, c = 1, 2, 3

# Print variables
print(x, y, z)  # Output: 5 Hello 3.14
