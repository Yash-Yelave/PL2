CSS Introduction

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Example:

body {
    background-color: lightblue;
}

h1 {
    color: white;
    text-align: center;
}

p {
    font-family: verdana;
    font-size: 20px;
}

CSS Syntax

A CSS rule consists of a selector and a declaration block. The selector points to the HTML element you want to style, and the declaration block contains one or more declarations separated by semicolons.

Example:

selector {
    property: value;
    property: value;
}

CSS Selectors

CSS selectors are used to select the HTML elements you want to style. There are several types of selectors including element, class, and ID selectors.

Example:

/* Element selector */
p {
    color: red;
}

/* Class selector */
.intro {
    font-weight: bold;
}

/* ID selector */
#header {
    background-color: blue;
}
